# bunaiCTF-2022
2022年12月23日の東京高専プロコンゼミ，ロボコンゼミ合同のクリスマスLTで開催したCTF．初心者がCTFerを増やすために実施しました．


難易度はこのCTF内のみで比較したものです．超入門者向けなのでほかの大会とかの`easy`位の問題が`medium`とか`hard`位の難易度だったりします．
